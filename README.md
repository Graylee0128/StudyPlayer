# 📚 StudyPlayer - 讀書戰鬥夥伴

這是一個專為考生設計的讀書排課與進度管理系統。透過直觀的介面與自動排課功能，幫助你更有效地分配各科目的學習時間，並隨時掌握目前的進度狀態。

## 🌟 核心功能

- **📊 進度管理**：預設多門考研/考公熱門科目（演算法、作業系統、離散數學等），以「半堂」為單位精準掌控剩餘份量。
- **📅 自動排課**：設定偏好的「星期」與「時段」（早、中、晚），系統會自動幫你填滿未來的空閒時間。
- **📱 互動行事曆**：左右滑動查看近三週的行程。
- **✅ 任務確認**：
    - **完成 (✔️)**：扣除課程庫存，邁向目標！
    - **取消 (❌)**：任務失敗將自動歸還時段至庫存，方便之後重新安排。
- **💾 離線儲存**：所有資料均存於瀏覽器的 `localStorage`，重新整理網頁也不會丟失進度。

## 🎨 設計風格

採用類 Duolingo 的明亮配色與圓角設計，讓枯燥的排課過程也像玩遊戲一樣充滿回饋感。

## 🚀 如何開始

### 方法 A：直接開啟
1. 直接開啟 `index.html` 即可開始使用。

### 方法 B：使用 Python Flask 伺服器
1. 確保已安裝 Python。
2. 安裝 Flask：
   ```bash
   pip install flask
   ```
3. 啟動伺服器：
   ```bash
   python app.py
   ```
4. 在瀏覽器開啟 `http://localhost:5000`。

## 📖 操作指南
1. 切換至 **[排課設定]** 分頁。
3. 選擇科目、偏好星期與時段。
4. 輸入欲安排的數量（單位：半堂）。
5. 點擊 **[確認排入行程]**，系統將自動填寫至 **[我的行程]**。
